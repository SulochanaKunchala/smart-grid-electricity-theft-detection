# Smart Grid Electricity Theft Detection using Machine Learning

## Project Overview
This project detects electricity theft in smart grid systems using Machine Learning.  
The model analyzes electricity consumption patterns and predicts whether electricity theft has occurred.

## Technologies Used
- Python
- Machine Learning
- Random Forest Classifier
- Streamlit Dashboard
- Pandas
- Scikit-learn
- Matplotlib

## Dataset Features
The dataset contains the following attributes:

- consumer_id
- avg_daily_usage_kwh
- peak_usage_kwh
- night_usage_kwh
- voltage
- current
- power_factor
- anomaly_score
- theft_label

## Machine Learning Model
The model used for prediction:

Random Forest Classifier

The model is trained on electricity usage data to detect abnormal consumption patterns.

## System Workflow

1. Load dataset
2. Train machine learning model
3. Accept user input
4. Predict electricity theft
5. Display results with graphs

## Output

The system shows:

- Electricity Theft Prediction
- Data Visualization Graphs
- Model Evaluation Metrics

## Deployment

The project is deployed using **Streamlit Cloud**.
